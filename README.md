# 🚀 Projetos em Destaque | Featured projects

### Hub Flow

Plataforma de **gestão ministerial modular**.

Features:

* sistema de módulos dinâmicos
* form builder
* visões em **tabela, kanban e calendário**
* PWA
* notificações
* customização por ministério

Stack:

```
Vue
Node.js
MongoDB
PWA
Docker
```

---

### LJ Cookies

Sistema completo de gestão para **produção e vendas de cookies artesanais**, desenvolvido para uso real em uma confeitaria.

Features:

* controle de estoque
* registro de vendas
* dashboard financeiro
* sistema de notificações
* módulo de compras
* sugestão de receitas baseada no estoque
* PWA

Diferencial:

* Sistema 100% customizável, permitindo adaptar módulos e regras de negócio conforme o crescimento da empresa.

Stack
```
Vue
Node.js
MySQL
PWA
Docker
```
